# CLI 参考
## 全局参数（每行含说明）
| 参数 | 说明 |
| --- | --- |
| `--dev` | 启用开发模式。 |
| `-g, --global` | 指定全局配置目录。 |
| `--json` | 以 JSON 形式输出。 |
| `--no-color` | 禁用彩色输出。 |
| `-p, --profile` | 选择 profile。 |
| `--plain` | 纯文本输出。 |
| `--update` | 检查更新。 |
| `-v, --version` | 输出版本信息。 |
| `-w, --workspace` | 指定工作区路径。 |

## 命令入口（每行含说明）
| 命令 | 说明 |
| --- | --- |
| `openclaw setup` | 初始化 OpenClaw 配置与工作区。 |
| `openclaw onboard` | 运行 CLI 引导流程，完成基础配置。 |
| `openclaw configure` | 打开交互式配置向导。 |
| `openclaw config` | 打开配置向导或执行 get/set/unset。 |
| `openclaw doctor` | 运行健康检查并给出修复建议。 |
| `openclaw status` | 查看当前状态摘要。 |
| `openclaw health` | 执行健康检查，支持 JSON 输出。 |
| `openclaw gateway` | 网关服务管理入口。 |
| `openclaw logs` | 查看并跟踪日志。 |
| `openclaw channels` | 通道管理入口。 |
| `openclaw message` | 消息发送与通道交互入口。 |
| `openclaw agent` | 运行单个代理任务。 |
| `openclaw agents` | 管理代理配置与身份。 |
| `openclaw sessions` | 会话列表与状态查看。 |
| `openclaw models` | 模型管理、别名与回退配置。 |
| `openclaw browser` | 浏览器自动化能力入口。 |
| `openclaw cron` | 定时任务管理入口。 |
| `openclaw plugins` | 插件管理入口。 |
| `openclaw skills` | 技能列表与校验入口。 |
| `openclaw approvals` | 执行审批与 allowlist 管理。 |
| `openclaw sandbox` | 沙箱环境管理入口。 |
| `openclaw docs` | 文档检索入口。 |
| `openclaw dashboard` | 打开控制台界面。 |
| `openclaw update` | 更新 CLI/组件。 |
| `openclaw reset` | 重置配置或缓存。 |
| `openclaw uninstall` | 卸载 OpenClaw。 |

> 更详细的参数和示例请查看“命令大全”。
